## Self learning project

- Forum App

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# Run Docker-Compose for development mode
$ yarn docker:up

# Terminate Docker-Compose
$ yarn docker:down

# development
$ yarn start

# watch mode
$ yarn start:dev

# production mode
$ yarn start:prod
```

## Test

```bash
# unit tests
$ yarn test

# e2e tests
$ yarn test:e2e

# test coverage
$ yarn test:cov
```
